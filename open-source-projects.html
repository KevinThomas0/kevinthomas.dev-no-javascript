Open Source Projects

Alpine Linux Package Maintainer

I maintain several packages for the Alpine Linux distribution, one of my favorite Linux distributions. I have contributed several aports to their repositories. For a list of packages I maintain, you can go here: https://pkgs.alpinelinux.org/packages?name=&branch=edge&maintainer=Kevin+Thomas

CryptoScamDB-Lists

Various lists in plain text format from CrytoScamDB to be used with services such as Pi-hole. The repository is hosted on my Gitlab.

kde-virtual-desktops-polybar

kde-virtual-desktops-polybar is a collection of scripts meant to be used with the KDE Plasma desktop environment in modules for Polybar - a status bar for linux desktops. The repository is hosted on my Gitlab.
